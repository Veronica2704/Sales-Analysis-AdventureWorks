# Sales-Analysis-AdventureWorks

## Objetivo del proyecto
Se tiene como objetivo analizar el desempeño de ventas de una empresa ficticia (AdventureWorks) esto para identificar tendencias, oportunidades de mejora y apoyar a la toma de decisiones del negocio.

## Preguntas del negocio
¿Cómo ha ecolucionado las ventas a lo largo del tiempo?
¿Qué productos generan mayor ingreso y utilidad?
¿Qué categorías son más rentables?
¿Qué regiones o países tienen mejor desempeño en ventas?
¿Quiénes son los clientes más valiosos?
¿Existen productos con alto volumen de ventas pero baja rentabilidad?

## Herramientas utilizadas
SQL para análisis y consultas.
Power BI para visualización.
Excel para validación y exploración inicial.
GitHub para la documentación de proyecto.
