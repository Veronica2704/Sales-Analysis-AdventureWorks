# Retail Sales Analysis Project

## Objetivo del proyecto
Se tiene como objetivo analizar los datos de ventas de una empresa de retail para entender el comportamiento de las ventas, la rentabilidad y el desempeño por producto, región y canal de venta.

## Preguntas del negocio
¿Cuáles son los productos más vendidos y más rentables?
¿Qué regiones generan mayores ingresos?
¿Cómo evoliconan las ventas en el tiempo?
¿Qué retailes tienen mejor desempeño?
¿Qué canal de venta es más eficiente?
¿Qué productos tienen alto columen pero baja rentabilidad?

## Dataset
9,649 registros
13 columnas
Fuente: Kaggle (Adidas Sales Dataset)

## Herramientas utilizadas
SQL para análisis y consultas.
Power BI para visualización.
Excel para validación y exploración inicial.
GitHub para la documentación de proyecto.
